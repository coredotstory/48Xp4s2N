4.15 총선 - 만 18세 유권자의 영향력은?
=====

2019년 12월 27일 선거연령을 만 18세로 낮추는 공직선거법 개정안이 통과됐다. 
2005년 만 19세로 선거연령 하향 이후 만 18세로 낮아진 선거연령이 2020년 4월 15일 제 21대 국회의원 선거에서 어떤 결과를 가져올지 탐구해보자.
